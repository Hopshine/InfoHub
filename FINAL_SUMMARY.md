## 🎯 使用方式

### 方式1: Web界面（推荐）

```bash
# 启动Web服务器
python web_app.py

# 浏览器访问
http://localhost:5000
```

**优点**:
- 🎨 可视化界面，操作直观
- 📊 实时数据统计
- 🔍 强大的搜索和过滤

