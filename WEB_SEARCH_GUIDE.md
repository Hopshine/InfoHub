# 🔍 Web界面搜索采集详细教程

## 📍 当前状态

- ✅ Web服务器运行中
- 🌐 访问地址: http://localhost:5000
- 📊 当前数据: 3篇演示文章

## 🎯 搜索采集步骤详解

### 第1步: 打开Web界面

在浏览器中访问：
```
http://localhost:5000
```

你会看到：
- 顶部统计卡片（文章总数、已分析等）
- 操作按钮栏
- 文章列表

### 第2步: 点击采集按钮

在操作按钮栏找到并点击：
```
📥 采集文章
```
这是一个橙色的按钮，位于"刷新数据"和"批量分析"之间。

### 第3步: 切换到搜索采集

在弹出的模态框中，你会看到3个标签页：
1. 单个URL
2. 批量URL
3. **搜索采集** ← 点击这个

### 第4步: 输入搜索参数

**搜索关键词输入框**：
```
输入: Claude
或: AI日报
或: GPT
或: 大模型
```

**最多采集数量输入框**：
```
建议: 5
范围: 1-20
```

### 第5步: 开始搜索采集

点击 **"搜索并采集"** 按钮

系统会：
1. 显示"搜索并采集中..."
2. 在搜狗微信搜索关键词
3. 解析搜索结果
4. 逐个采集文章内容
5. 保存到数据库
6. 显示采集结果

### 第6步: 查看结果

采集完成后会显示：
```
搜索采集完成
总计: X | 成功: Y | 失败: Z

✓ 文章标题1
✓ 文章标题2
✗ 文章标题3 (已存在)
```

## ⚠️ 可能遇到的问题

### 问题1: 遇到验证码

**现象**：
```
✗ 搜索采集失败: 未搜索到文章或遇到验证码
```

**原因**：
- 搜狗微信有反爬机制
- 频繁请求触发验证码

**解决方案**：
1. **使用批量URL导入**（推荐）
2. 配置代理IP
3. 减少采集数量
4. 增加时间间隔

### 问题2: 搜索无结果

**现象**：
```
✗ 搜索采集失败: 未搜索到文章
```

**原因**：
- 关键词太具体
- 关键词拼写错误

**解决方案**：
- 使用更通用的关键词
- 尝试不同的关键词组合

### 问题3: 采集失败

**现象**：
```
✗ 文章标题 (获取内容失败)
```

**原因**：
- 文章已删除
- URL已过期
- 网络问题

**解决方案**：
- 检查网络连接
- 稍后重试
- 使用其他文章URL

## 🎯 推荐的采集方式

### 方式1: 批量URL导入（最稳定）⭐⭐⭐⭐⭐

**步骤**：
1. 手动访问 https://weixin.sogou.com/
2. 搜索关键词（如"Claude"）
3. 复制文章URL
4. 在Web界面选择"批量URL"
5. 粘贴URL（每行一个）
6. 点击"批量采集"

**优点**：
- ✅ 最稳定
- ✅ 不会遇到验证码
- ✅ 成功率高

### 方式2: 单个URL采集 ⭐⭐⭐⭐

**步骤**：
1. 获取微信文章URL
2. 在Web界面选择"单个URL"
3. 粘贴URL
4. 点击"开始采集"

**优点**：
- ✅ 简单快速
- ✅ 适合测试

### 方式3: 搜索采集 ⭐⭐⭐

**步骤**：
1. 在Web界面选择"搜索采集"
2. 输入关键词
3. 点击"搜索并采集"

**优点**：
- ✅ 自动发现文章
- ⚠️ 可能遇到验证码
